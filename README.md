Jogo de Adivinhar Cores
Um simples e viciante jogo de navegador onde o objetivo é clicar na cor correta o mais rápido possível para acumular o máximo de pontos e entrar para o ranking dos 10 melhores.

 Regras do Jogo

    Esta versão do clássico jogo de adivinhar cores possui algumas regras e características específicas:

    Objetivo: Clicar no quadrado que possui a mesma cor do círculo de referência exibido no topo da área do jogo.

    Tempo Limite: Cada partida tem uma duração fixa de 40 segundos. O objetivo é fazer o máximo de pontos dentro desse tempo.

    Pontuação: O sistema de pontuação foi ajustado para recompensar a precisão e velocidade:

    Acerto: +20 pontos.

    Erro: -10 pontos de penalidade.

    Ranking: O jogo salva automaticamente as 10 maiores pontuações no navegador do usuário (localStorage). Ao final da partida (quando o tempo se esgota), o jogador é convidado a inserir seu nome para registrar sua pontuação.

    Fim de Jogo: A partida termina exclusivamente quando o temporizador chega a zero.

Como Jogar
Abra o arquivo index.html em seu navegador.

    Clique no botão "Jogar" para iniciar a partida e o temporizador.

    Observe a cor no círculo de referência.

    Clique no quadrado correspondente entre as 9 opções coloridas.

    Continue jogando e acumulando pontos até o tempo acabar.

    Ao final, digite seu nome na caixa de diálogo para salvar sua pontuação no ranking.

Como Executar
Este é um projeto simples de front-end e não requer instalação de dependências ou um servidor web.

    Faça o download ou clone este repositório.

    Certifique-se de que os arquivos index.html, style.css (ou o nome do seu arquivo de estilo) e script.js estejam na mesma pasta.

    Abra o arquivo index.html diretamente em um navegador de internet moderno (Google Chrome, Mozilla Firefox, Microsoft Edge, etc).
